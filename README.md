#  ToDo List on Play 2.5.5 (Java)
Proof of concept for ToDo Web Site with RESTful interface.

## Used plugins
- Ebean with H2 in memory database
- Bootstrap 3.3.7
- RequireJS 2.2.0
- jQuery 3.1.0

## Autotests
- JUnit
- FluentLenium
- Rest-assured
